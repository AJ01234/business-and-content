# Failure Archaeology

Written 2026-07-12. Ground truth: `context/past-attempts.md`, `context/track-record.md`,
`goal.md` ("What this goal is NOT"), archive `# 12-Month Goal.md`, interviews July 2026.

## Tried and failed (with the actual mechanism)
1. **HybridBanda run #1** — 120+ videos, views 130–500, stopped from low-view exhaustion.
   Mechanism: no recognizable format/series/protagonist-lens; NOT a lack of effort.
   Banned response: "post more consistently" lectures. Correct response: format + hooks
   (see quality-bar skill).
2. **Chronic underpricing** — the documented ladder: ₹80 quoted for a project, ₹800
   banner, half-paid editing, ₹500–700/video retainer (25–30 videos for ₹15–20K).
   Mechanism: fear + no written floor. Fix that exists: price floors in context/offer.md.
   Any plan that "wins the client by going cheaper" repeats this failure.
3. **Research/planning instead of selling** — his own bad-week definition: "consuming
   content and planning instead of selling or pitching or doing things." Multi-year
   pattern (Master_Context). Any deliverable that is another plan/document instead of a
   pitch, video, or invoice should be challenged — **including work on this repo and
   this manual**: polishing goal.md, context/, or skills counts as planning, not
   progress.
4. **Identity switching** — creator → agency owner → AI builder → operator, repeatedly
   (Master_Context). Fix: goal.md Decision Rule 5, one experiment at a time.

## Deliberately rejected (do NOT re-propose without new evidence)
1. **₹3,00,000/month take-home as the 12-month number** — he admitted it was picked, not
   derived (interview Q1). Current derived target: ₹1L/month (goal.md §2).
2. **RR Garden as a co-equal goal** — demoted to Client Zero, 6–8h/week cap. Reason: only
   4 operable rooms, ~6–10 weddings/yr, one semi-manager; family expectation is real but
   bounded (context/rr-garden.md).
3. **Wedding-media agency + cofounder** — archive doc scored "starting a permanent agency
   immediately 3/10, adding a cofounder before validation 2/10"; RR Garden's 6–10
   weddings/yr is below even the archive's weak-case volume. Revisit ONLY after ≥5
   packages sell via partner vendors.
4. **Leading with "growth systems"** — zero paid instances ever (track-record.md).
   Upsell-only, gated by goal.md Decision Rule 3.
5. **Topic niche for HybridBanda** — explicitly rejected by him ("I don't actually want a
   niche"). The lens is fixed; the topics are not.
6. **Building software/tools before validation, new social pages before 3–5 paying
   customers, full-time hires** — all barred in goal.md Constraints.
7. **The aesthetic-clinic JOB** — offered, declined (track-record.md). Employment is the
   runway-rule fallback (Decision Rule 1), not the plan.

## The meta-rule
Before endorsing any "new" direction, search this file and goal.md's NOT-list first. If
it appears here, the burden of proof is NEW EVIDENCE (recorded conversations, paid
pilots), never enthusiasm.

## When NOT to use this skill
Don't use it to kill honest experiments that are INSIDE the current offer (new hook
styles, new pitch scripts, new client segments within ICP) — it guards direction, not
iteration.

How to re-verify: at each 90-day review, ask "did anything on the rejected list earn new
evidence?" and move items in/out with a dated note.
