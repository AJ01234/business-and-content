# Repo Map — Orientation for a zero-context operator

Written 2026-07-12. Source of truth: the files themselves.

## What this project is
One person — **Atishay**, a creator/operator in Kanpur, India — building toward
**₹1,00,000/month personal take-home income by July 11, 2027** (held 3 consecutive
months), by selling productized video retainers, documented publicly on his channel
**HybridBanda**. The repo is his externalized brain: goal, context, systems.

## Reading order (do not skip)
1. `goal.md` — THE operating goal: outcome, numbers, deadline, constraints, decision
   rules, known gaps. Everything else serves this file.
2. `context/README.md` — index of the 16 context files (who he is, clients, money,
   market, quality bar…). Read the specific file for any topic before acting on it.
3. `system/README.md` — the weekly reel production system (his 2-shorts/week commitment);
   the same folder also holds the two sales assets: `offer-sheet.md` and
   `message-templates.md`.
4. `.claude/skills/` — this manual.
5. ARCHIVE ONLY: `# 12-Month Goal.md` (superseded multi-goal draft) and
   `Master_Context.md` (a 53k-line ChatGPT export about niche discovery). Mine for
   history; never treat their instructions as current. goal.md says so explicitly.

## Glossary (define-every-term rule)
- **HybridBanda** — Atishay's personal Hinglish creator channel (Instagram/YouTube).
  ~120+ videos published in a first run that plateaued at 130–500 views (context/past-attempts.md).
- **RR Garden** — his father's guest house + wedding venue in Kanpur. In this project it
  is **"Client Zero"**: the practice client, capped at 6–8 h/week (context/rr-garden.md).
- **Desire Edits** — an editing shop Atishay works with informally (unpaid-ish, no
  written terms). NOT income, NOT confirmed capacity (context/desire-edits.md).
- **Warm threads** — the ENTIRE warm network: 5 threads listed in
  context/sales-pipeline.md (Dipankar, Reshma, aesthetic-clinic manager, Abhishek's
  circle, Palava WhatsApp group). Confirmed complete July 2026.
- **The retainer** — the entry offer: monthly short-form + AI-video package,
  floor ₹15,000/month (context/offer.md).
- **Growth systems** — CRM/follow-up/lead-tracking service. UPSELL ONLY; nobody has ever
  paid for it (goal.md Decision Rule 3).
- **Qualified conversation** — decision-maker + their content/leads discussed +
  past-spend question asked + concrete next step proposed (context/sales-pipeline.md).

## Maintenance rules (from context/README.md)
One topic per file; date significant updates; label unverified assumptions; after every
20 qualified conversations revisit offer/ICP/pricing files against recorded objections.
⚠️ **Meta-work tripwire:** time spent editing this repo is PLANNING, not selling or
shipping — it counts toward the bad-week signal (context/operating-rhythm.md), never
against it. If repo edits outnumber pitches+videos in a week, the documented avoidance
pattern is running.

## When NOT to use this skill
Don't re-read the archive files for operational answers; don't use this skill as a
substitute for the topic-specific skill (pricing, pipeline, reels…) once oriented.

How to re-verify: `ls` the repo and diff this map against actual files; read goal.md's
Known Gaps table for what has since closed.
