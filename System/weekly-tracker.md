## Rules
- "My cycle minutes" = actual research+scripting involvement. Target ≤20. If it creeps
  past 30 two weeks running, the leak is almost always re-running prompts or re-picking
  ideas — both are banned by the system, so name which one happened.
- Views column gets filled ONCE, Sunday. No mid-week checking.
- Format verdicts only at post #50 (past-attempts.md). Until then the only pass/fail is:
  did 2 reels ship?
- Every 4 weeks: move one lesson into the relevant context/ file so the brain folder
  stays current (context/README.md maintenance rule).
