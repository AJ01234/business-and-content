# Shoot Checklist — 30-minute box per reel

Gear (tools.md): S25 Ultra + ₹2K mic + tripod. Nothing else. If a shot needs gear you
don't own, it's the wrong shot.

## Before pressing record (5 min)
- [ ] Script on phone/printed — shot list visible
- [ ] Mic connected + 5-second test clip played back
- [ ] 4K 30fps (or 60 for action), vertical, exposure locked on face
- [ ] Clean the lens (every time — this is the free quality upgrade)
- [ ] Location matches the story (RR Garden shots AT RR Garden — real places, real stakes)

## Recording (20 min hard cap)
- [ ] Hook first, 3 takes MAX — pick the most alive one, not the cleanest
- [ ] Each beat: 2 takes max. Flubbed line? Snap fingers, repeat the line, keep rolling
- [ ] Proof moment: capture the actual thing (register, dashboard, cash, screen)
- [ ] B-roll last: 6 shots × 10–15 seconds each, from the research list

## The ADHD tripwires (operating-rhythm.md)
- Timer ON, 30 minutes, visible. When it rings: you have the video. Stop.
- "One more take" after take 3 = the perfect-shot spiral. The retake never wins by enough
  to matter at 480p compression on a phone screen.
- Cringe is allowed (Q9: you're comfortable). Boring is the only failure.

## After (2 min)
- [ ] Scrub through takes once — flag keepers with a screenshot
- [ ] Files into this week's folder (one folder per reel, named `S1-ep01-guesthouse`)
