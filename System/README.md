# HybridBanda Weekly Reel System

> Also in this folder (NOT part of the reel cycle): `offer-sheet.md` (printable client
> one-pager) and `message-templates.md` (outreach/follow-up texts) — they belong to the
> sales pipeline (context/sales-pipeline.md), added 2026-07-12.

Target: **~20 minutes of Atishay's involvement per cycle** for research + scripting of the
week's 2 reels. Shooting and editing stay human (he's the protagonist and the editor) but
get timeboxed checklists so they can't spiral.

## The current process, mapped (reconstructed — no process.md/time.md existed)

| # | Step (run #1 behaviour) | Old cost | Verdict |
|---|---|---|---|
| 1 | Consume content "for inspiration" | Hours, unbounded | **AUTOMATE** → idea-bank.md kills doomscroll-as-research |
| 2 | Decide what to make (mood-driven, re-litigated) | 30–60 min | **TEMPLATE** → pick top idea from pre-scored bank |
| 3 | Research topic/hook (perfectionist rabbit hole) | 1–3 hrs | **AUTOMATE** → research-prompt.md, one AI run |
| 4 | Script from blank page (chase perfect script) | 1–2 hrs | **AUTOMATE draft + HUMAN punch-up** → script-template.md |
| 5 | Shoot (chase perfect shot) | Unbounded | **KEEP HUMAN + TEMPLATE** → shoot-checklist.md, 30-min box |
| 6 | Edit (the 5-day spiral) | Up to days | **KEEP HUMAN + TEMPLATE** → edit-checklist.md, 2 sessions max |
| 7 | Caption/publish (ad hoc) | 20–30 min | **TEMPLATE** → publish-checklist.md |
| 8 | Review performance (obsess or skip) | Varies | **TEMPLATE** → weekly-tracker.md, 5 min Sunday |

Why the blank page is the enemy: profile.md (ADHD → perfection spiral) and past-attempts.md
(120 videos, packaging problem). Every step here starts from a filled template, never blank.

## The new cycle — where the 20 minutes go

Run once weekly (pair it with the Sunday scorecard from sales-pipeline.md):

| Step | Who | Atishay's time |
|---|---|---|
| 1. Open idea-bank.md, pick 2 ideas for the week | Atishay | **5 min** |
| 2. Run research-prompt.md (paste + fill 3 blanks) into any AI | AI | 0 (runs while he does something else) |
| 3. Run script pass: AI fills script-template.md for both ideas | AI | 0 |
| 4. Read both drafts, punch up in HIS voice (hooks + one line of himself per script) | Atishay | **12 min** |
| 5. Log the week's row in weekly-tracker.md | Atishay | **3 min** |

**Total: ~20 min.** Then, separately: shoot (30-min box per reel), edit (2 sessions max),
publish (10-min checklist).

## Hard rules (from context/, non-negotiable)
- Ship beats perfect: fix the ONE biggest gap, publish (quality-bar.md)
- Never judge by views before ~50 posts of this format (past-attempts.md)
- Ideas outside the 3 series go INTO the bank, not into production (goal.md Rule 5)
- If a cycle is missed, the next cycle picks the SAME ideas — no re-deciding
