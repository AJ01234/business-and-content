# Edit Checklist — 2 sessions max, then it ships

Tools: CapCut for speed (default), Resolve when the reel earns it (S3 passion pieces).
The 5-day edit killed run #1. This SOP exists so it can't happen again.

## Session 1 — Assembly (45 min cap)
- [ ] Drop keeper takes in script order — no trimming yet (10 min)
- [ ] Rough cut to spine: hook / setup / friction / proof / button (20 min)
- [ ] Kill every pause >0.5s; jump cuts are the style (quality-bar: Neistat/Mayya pace)
- [ ] Proof moment insert on screen
- [ ] Watch once at 1x. Note the ONE biggest gap. Stop session.

## Session 2 — Polish (45 min cap, different day is fine)
- [ ] Fix the ONE gap from session 1 — nothing else structural
- [ ] Captions/subtitles (auto + quick correct — Hinglish needs manual pass on Hindi words)
- [ ] Sound: music under voice, duck it during spoken lines; one SFX max per beat
- [ ] Color: one preset/LUT applied globally. NO per-clip grading on a reel
- [ ] Export 4K vertical. Done means EXPORTED.

## Ship gate (quality-bar.md checklist, 60 seconds)
1. Protagonist doing something real? 2. Story spine present? 3. Edit moves? 4. One moment
of substance? 5. Fix-one-gap-and-ship applied?
→ All yes = publish today via publish-checklist.md. Do NOT rewatch more than twice after
export; each extra rewatch only manufactures doubt (past-attempts.md lesson).

## Absolute rule
A reel never gets a third session. If it truly collapsed, it goes back to the idea bank
with a note — that's a research failure, not an editing one.
